(static) apiCall(endpoint, options) → {object}

Prepares and sends a raw request to the api server.
Parameters:
Name Type Description
endpoint string The requested endpoint.
options object The option object. Possible parameters:
Properties
Name Type Description
callback int Required on client side.
ids string | int | Array.<int> | Array.<string> The requested ids. Not available on endpoints without id in its result set. Either an array of ids or a comma-separated list of ids or a single id. Depending on the endpoint, it's also possible to provide alphanumerical ids such as GUIDs.
pageIndex int Will request a specific page of a result set for endpoints with bulk enabled requests.
pageSize int The size of the requested pages. Needs to be set with options.pageIndex.
key string The authentication key that grants access to the requested endpoint. Only needed for authenticated endpoints.
language string Only available for endpoints with language awareness. Defines the requested language. The API server will return the language that your browser provides when language is omitted.
quantity int Only available for endpoint v2/commerce/exchange. See GW2API.getCommerceExchange.
input int Only available for endpoint v2/recipes/search. See GW2API.searchRecipes.
output int Only available for endpoint v2/recipes/search. See GW2API.searchRecipes.
