staticGW2API.apiCall(endpoint, options, callback){object}

E:/Projekte/Guild Wars 2/guild-viewer/packages/gw2api/gw2api.js, line 1060
Prepares and sends a raw request to the api server.
Name Type Description
endpoint string The requested endpoint.
options object optional The option object. Possible parameters:
Name Type Description
ids string | int | Array.<int> | Array.<string> optional The requested ids. Not available on endpoints without id in its result set. Either an array of ids or a comma-separated list of ids or a single id. Depending on the endpoint, it's also possible to provide alphanumerical ids such as GUIDs.
pageIndex int optional Will request a specific page of a result set for endpoints with bulk enabled requests.
pageSize int optional The size of the requested pages. Needs to be set with options.pageIndex.
key string optional The authentication key that grants access to the requested endpoint. Only needed for authenticated endpoints.
language string optional Only available for endpoints with language awareness. Defines the requested language. The API server will return the language that your browser provides when language is omitted.
quantity int optional Only available for endpoint v2/commerce/exchange. See GW2API.getCommerceExchange.
input int optional Only available for endpoint v2/recipes/search. See GW2API.searchRecipes.
output int optional Only available for endpoint v2/recipes/search. See GW2API.searchRecipes.
callback function optional A callback that will be called after the API sends a response. The function will get two parameters error and result. See HTTP.call() for details.
Throws:
Meteor.Error This function will throw any HTTP.call() error untouched unless you override the error handling with GW2API.setErrorHandler.
Returns:
result object of the HTTP request. See HTTP.call() for details.
